## Introduction

This report presents the process of implementing tools at Abstergo Industries, carried out by Luiz Felipe Rodrigues de Freitas. Abstergo Industries operates as a pharmaceutical reseller, distributing products to other pharmacies, and currently has no cloud infrastructure in place.

The goal of this project was to identify 3 AWS services with the purpose of achieving immediate cost reduction, taking into account the company's profile and the financial management team's low familiarity with cloud technology.

---

## Project Description

The tool implementation project was divided into 3 stages, each with its own specific objectives. The stages are described below:

### Stage 1:
- **Tool name:** Amazon S3 (Simple Storage Service)
- **Tool focus:** Secure, scalable, and low-cost storage for company data and documents.
- **Use case description:** Abstergo Industries handles large volumes of documents on a daily basis: invoices, purchase orders, contracts with partner pharmacies, and product records. These files are currently stored on local physical servers with high maintenance costs and risk of data loss. With Amazon S3, all documentation will be migrated to the cloud with on-demand storage — the company pays only for what it uses — eliminating fixed hardware costs and ensuring automatic data availability and redundancy.

### Stage 2:
- **Tool name:** Amazon RDS (Relational Database Service)
- **Tool focus:** Managed cloud database for inventory control, orders, and customer management.
- **Use case description:** Managing inventory and tracking orders between Abstergo Industries and its pharmacy clients requires a reliable, always-available database. With Amazon RDS, the company replaces locally installed databases — which demand expensive licenses, manual backups, and specialized staff — with a fully managed AWS solution that includes automatic backups, security updates, and scalability as the business grows.

### Stage 3:
- **Tool name:** Amazon SNS (Simple Notification Service)
- **Tool focus:** Notification and messaging service for automated communication with partner pharmacies.
- **Use case description:** Currently, communication between Abstergo Industries and its pharmacy clients is handled manually (phone calls, individual emails), leading to rework and unnecessary operational costs. With Amazon SNS, the company can automate notifications for order confirmations, delivery updates, and critical stock alerts — charging only per message sent, with no fixed infrastructure required.
